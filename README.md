# online-superMarket
sever side:node.js, client side:angular.
Online supermarket. Here you can buy all the groceries you need at home

1. git clone my repository.

2. open new terminal.

3. cd dev-apps.

4. docker compose up.

5. open new terminal.

6. cd apps\server.

7. npm install.

8. npm run all-slim.

9. open new terminal.

10. cd apps\client.

11. npm install.

12. npm start.
